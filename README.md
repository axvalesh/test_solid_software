# Test task for Solid Software

This repository contains my implementation of the Flutter test task for Solid Software.

## Original task

The application should:

* display the text **"Hello there"** in the middle of the screen
* change the background color after tapping anywhere on the screen
* generate colors using RGB without external libraries for color generation

## Improvements

In addition to the original requirements, I extended the solution with the following improvements:

* refactored the logic to **BLoC/Cubit**
* added **color history**
* added the ability to **return to previously selected colors**
* added the ability to **enter a HEX value** and apply a custom color
* improved the overall **maintainability** and **scalability** of the solution
